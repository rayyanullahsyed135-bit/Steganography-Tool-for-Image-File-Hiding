# Steganography-Tool-for-Image-File-Hiding

## 📌 Overview
This project is a Python-based steganography tool that allows users to hide and extract secret messages inside images using the Least Significant Bit (LSB) technique.

## 🚀 Features
- Hide text inside images
- Extract hidden messages
- GUI interface using Tkinter
- Supports PNG and BMP formats
- Optional encryption (future scope)

## 🛠️ Technologies Used
- Python
- PIL (Pillow)
- Tkinter
- Stepic (optional)
